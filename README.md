# Personal Portfolio 🚀

A minimal, high-performance personal portfolio built with Next.js, React, Tailwind CSS, TypeScript, and Shadcn UI. This site features a custom Bento Grid layout to showcase technical skills and an interactive modal framework for deep-diving into engineering projects.

🔗 **Live Demo:** [personal-portfolio-three-amber-63.vercel.app](https://personal-portfolio-three-amber-63.vercel.app/)

---

## 🛠️ Tech Stack

*   **Framework:** Next.js 14 (App Router)
*   **Language:** TypeScript
*   **Styling:** Tailwind CSS
*   **UI Components:** Shadcn UI & Lucide React Icons
*   **Hosting & Deployment:** Vercel (CI/CD Pipeline)

---

## ✨ Features

*   **Bento Grid Architecture:** A modern layout engineered to present technical competencies cleanly.
*   **Interactive Project Showcases:** Custom modular windows highlighting system architectures (including a detailed breakdown of a C++ Smart Emergency Response System).
*   **Smooth Client-Side Scrolling:** Optimized navigation that anchors seamlessly to deep-dive sections.
*   **Automated Production Pipeline:** Fully integrated GitHub-to-Vercel continuous deployment workflow.
